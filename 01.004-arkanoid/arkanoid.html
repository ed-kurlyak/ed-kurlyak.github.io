Игра FREAKOUT Remastered на С++ WinAPI Visual Studio 2019

Загрузить исходный код С++ WinAPI Visual Studio 2019 ЗДЕСЬ

Страница проекта на GitHub https://github.com/ed-kurlyak/freakout-remastered

Игра взята из книги Андре Ла Мота Tricks of the Windows Game Programming Gurus (1999), Программирование игр для Windows. Советы профессионала, 2-е издание. В книге автор реализовал игру на C++ и DirectDraw, игра в книге носит название FreakOut. В моем примере игра реализована на C++ WinAPI. Исходный код в книге был представлен в папке T3DCHAP01 и был разработан для старых версий Visual Studio и DirectDraw.
